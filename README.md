# Rikiki

Code qui permet de compter les points du jeu de cartes "Rikiki".

Il permet dans un premier temps de "monter" (augmenter le nombre de cartes de chacun des joueurs), 
puis demande à chaque tour si les joueurs veulent "descendre" ou bien "arrêter de jouer".

Si les joueurs décident d'arrêter de jouer, le code annonce le score.
Si les joueurs veulent redescendre, alors la partie continue jusqu'à arriver au tour 1, puis annonce ensuite le score.
